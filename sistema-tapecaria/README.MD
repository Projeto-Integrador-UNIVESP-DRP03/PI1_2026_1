# Repositório da aplicação web ZitOS

<h1>
  <img src="imagens\Identidade_visual\logo_aplicacao.jpeg" alt="Logo" style="height:1em; vertical-align:middle;">
  Sistema de Gerenciamento de Ordens de Serviço do Zito
</h1>

O objtivo da aplicação é informatizar os pedidos de personalização de estofados de automóveis, um dos principais serviços oferecidos pela [Zito Tapeçaria](https://zitotapecaria.com.br/tapecaria-e-vidracaria-em-pindamonhangaba/). Nela será registrado os dados dos clientes, veículos e dos pedidos orçados.

Com isso, espera-se melhorar o gerenciamento dos pedidos e com o armazenamento das informações, será possível fazer uma análise histórica dos pedidos e analisar o desempenho da empresa.
